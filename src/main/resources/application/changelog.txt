BirthdayManager Changelog
=========================

0.7.2
- Extract shared CSV file chooser defaults for open and save flows.
- Improve error logging context across controllers, tasks, and utilities.

0.7.1
- Localize remaining runtime dialogs, alerts, and labels.
- Fix and localize the week and month overview context menus.

0.7.0
- Add a shortcuts view that can be opened from About and Preferences.
- Show the bundled changelog directly in the About window.
- Keep popup windows from shrinking when switching to the shortcuts view.
- Improve keyboard navigation through the birthday edit forms.
- Skip duplicate birthdays while importing data.

0.6.2
- Fix edit and delete actions so they keep using the selected person.
- Keep the current month-week selection stable while refreshing the overview.
- Run the missed birthdays check only once per session.

0.6.0
- Add the month calendar view next to the existing week overview.
- Refresh birthday sublists after loading a file.
- Improve packaging and standalone JAR support on current Java releases.
- Improve search, leap-year birthday handling, and update reminder persistence.

Earlier highlights
- Add the About view and improve its usability.
- Allow double-clicking birthdays to open them.
- Improve the missed birthdays prompt and general file handling.
